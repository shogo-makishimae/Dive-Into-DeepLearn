## 🛰️ 核心实验场 (The Laboratory)

本项目通过从零实现（From Scratch）与框架实现（Framework-based）双路径，深度拆解 AI 的黑盒：

- 🧬 **数学原力**：从张量运算到自动求导，解析梯度的流动轨迹。
- 👁️ **视觉进化**：ResNet, DenseNet, ViT... 探索卷积与注意力机制的边界。
- 🗣️ **语言建模**：从 RNN 到 Transformer，重构理解语言的神经元。
- ⚡ **工业炼丹**：多 GPU 并行训练、学习率衰减策略与模型压缩。

## 🛠️ 装备舱 (Tech Stack)

| 维度 | 组件 | 描述 |
| :--- | :--- | :--- |
| **内核** | `Python 3.9+` | 所有的魔法都始于此 |
| **框架** | `PyTorch 2.0+` | 极致的灵活性与计算性能 |
| **数学** | `NumPy / Pandas` | 数据预处理的瑞士军刀 |
